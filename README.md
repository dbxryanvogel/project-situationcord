# Project Situationcord

A Turborepo monorepo managing multiple applications.

## Apps

- `apps/discord-server` - Discord bot/server application
- `apps/web` - Web application

## Getting Started

Install dependencies:

```bash
bun install
```

Run all apps in development mode:

```bash
bun dev
```

Build all apps:

```bash
bun build
```

## Using Turborepo

This monorepo uses [Turborepo](https://turbo.build/repo) for build orchestration and caching.

## Tech Stack

- **Package Manager**: Bun
- **Build System**: Turborepo
- **Apps**: Discord Server, Web
